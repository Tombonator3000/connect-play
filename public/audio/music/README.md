# Music Library

This folder contains ambient music tracks for Mythos Quest.

## Required Music Files

### Main Ambience
- `horror-ambient.ogg` - Main game background (loopable, 2-5 min)
- `menu-theme.ogg` - Main menu music
- `tension.ogg` - High tension/combat music
- `exploration.ogg` - Calm exploration theme

### Phase-Specific
- `mythos-phase.ogg` - Mythos phase ominous drone
- `combat-phase.ogg` - Combat encounter music
- `victory.ogg` - Victory fanfare (short, 10-20 sec)
- `defeat.ogg` - Defeat theme (short, 10-20 sec)

### Weather Ambience
- `rain-ambient.ogg` - Rain + thunder ambient loop
- `wind-ambient.ogg` - Howling wind loop
- `fog-ambient.ogg` - Eerie fog soundscape

## Style Guidelines

The game is set in 1920s Lovecraftian horror. Music should be:
- **Atmospheric** - Ambient drones, minimal melody
- **Dissonant** - Minor keys, tritones, unsettling harmonies
- **Period-appropriate** - Jazz influences, vintage instrumentation
- **Dynamic** - Builds tension, ebbs and flows
