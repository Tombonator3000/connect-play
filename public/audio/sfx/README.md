# Sound Effects Library

This folder contains sound effects for Mythos Quest.

## Required SFX Files (CC0/Royalty-Free)

### Combat Sounds
- `sword-swing.ogg` - Melee weapon swing
- `sword-hit.ogg` - Melee weapon impact
- `gunshot.ogg` - Firearm discharge
- `reload.ogg` - Weapon reload

### Magic/Spell Sounds

- `eldritch-bolt.ogg` - Offensive spell
- `heal.ogg` - Healing spell
- `banish.ogg` - Banishment spell
- `dark-energy.ogg` - Dark magic effect

## Recommended Free Sources

1. **OpenGameArt.org** - CC0 licensed game audio
   - https://opengameart.org/content/100-cc0-sfx
   - https://opengameart.org/content/80-cc0-rpg-sfx
   
2. **Freesound.org** - CC0 tagged sounds
   - Filter by "0" license for public domain
   
3. **Sonniss GDC Audio Bundles** - Free professional audio packs
   - Released annually, royalty-free

## File Format Notes
- Prefer `.ogg` format for web (smaller, good quality)
- `.mp3` also works but larger file size
- Keep files under 500KB each for fast loading
